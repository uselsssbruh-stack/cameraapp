# Camera App

A simple and easy-to-use camera application for Android devices.

## Features

- **Basic Camera Functionality**: Take photos with your device's camera
- **Photo Capture**: Easily capture moments with a single tap
- **Simple Interface**: User-friendly design for quick access to camera features
- **Android Native**: Built with native Android development using Kotlin and Gradle

## Requirements

- Android API Level 21 or higher
- Android Studio for development
- Gradle build system

## Installation

1. Clone or download this repository
2. Open the project in Android Studio
3. Connect your Android device or emulator
4. Click **Run** or press `Shift + F10` to build and install the app

## Usage

1. Launch the Camera App on your device
2. Point the camera at what you want to capture
3. Tap the capture button to take a photo
4. View or share your photos

## Project Structure

```
cameraapp/
├── app/                 # Main application module
├── gradle/             # Gradle wrapper files
├── build.gradle.kts    # Project build configuration
├── settings.gradle.kts # Gradle settings
├── gradle.properties   # Gradle properties
├── gradlew            # Gradle wrapper script
└── gradlew.bat        # Gradle wrapper for Windows
```

## Building

```bash
./gradlew build
```

## Running on Device

```bash
./gradlew installDebug
```
